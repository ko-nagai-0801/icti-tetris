# iOS DEVICE QA

最終更新: 2026-02-20

## 目的

iPhone実機 Safari / ホーム画面追加（PWA）で、MVPの主要導線が問題なく完了できるかを確認する。

## 事前準備

1. iPhone を同一ネットワークに接続
2. 開発PCで `npm run dev -- --host 0.0.0.0 --port 4173` を起動
3. iPhone Safari で `http://<PCのIP>:4173` を開く

## チェック項目

1. Home表示
   - 注意書きが表示される
   - セッション開始 / 記録 / 設定 / 中断・相談に遷移できる
2. セッション導入
   - 「すべてチェック」ボタンが動作
   - 開始で次ステップへ進む
3. 短い思い出し（再活性化）
   - タイマーが減る
   - スキップで次へ進む
   - 中断でホームへ戻る
4. 回転ミニ課題
   - 3問完了できる
   - 選択肢に重複がない
5. テトリス
   - タッチ操作（左右/回転/ドロップ/ホールド）が機能
   - 画面崩れなし（縦持ち）
6. セッション後チェック
   - スライダー入力→保存でホームへ戻る
7. ログ
   - 新規セッションが表示される
   - 詳細画面が開ける
8. 設定
   - 再活性化秒数変更が次セッションへ反映
   - 技術ログのコピー/削除が動作
9. PWA
   - 「ホーム画面に追加」後も起動できる
   - Safe Area（上部/下部）でUIが欠けない
10. オフライン
   - ネットワークOFF時に `/offline` 表示へ遷移
